# RehearsalLink 完了済みタスク

これまでに完了したフェーズの記録です。

## Phase 1: プロジェクト基盤とファイル読み込み
- 音声ファイルのロード、基本モデルの定義

## Phase 2: 波形表示と再生の基本
- 波形描画エンジン、再生制御 (Play/Pause/Seek)

## Phase 3: 自動判定ロジックの実装
- RMS/周波数解析による演奏・会話・無音の自動分類

## Phase 4: UIインタラクションと編集
- セグメントの選択、境界移動、分割、タイプ変更

## Phase 5: データ保存と仕上げ
- JSON形式でのプロジェクト保存・読み込み

## Phase 6: 機能拡張と出力強化
- 波形ズーム機能 (水平方向)
- Inspector表示更新 (セグメント詳細)
- ラベル入力フィールド追加
- セグメントの繰り返し再生
- セグメント結合、ファイル出力処理 (演奏のみ)
- セグメント結合、ファイル出力処理 (会話のみ)
- 会話セグメントの文字起こし機能 (SpeechAnalyzer)

## Phase 7: リリース準備
- アプリケーションアイコンの作成
- Productionビルドの実行
- .app バンドルの作成と配布用ZIPの作成
